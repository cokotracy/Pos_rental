SETUP GUIDE TECHNICAL

  • FIRST we setup Kitchen Screen, active sync between 2 sessions [ Kitchen -> Cashier/Waiter ]
  • Setup Cashier/Waiter Screen, active Floors and Tables for order
  • Setting Cashier/Waiter Config active sync between Session too [Waiter/Casher -> Kitchen]
  • Cashier/Waiter take order, click to button Order -> will create new Ticket send to -> Kitchen Screen
  • On Waiter/Cashier Screen you will have screen for management all tickets send to Kitchen Screen
  • All tickets send from Waiter/Cashier will Display on KITCHEN ROOM Screen. Kitchen users can management all tickets send from Waiter/Cashier
